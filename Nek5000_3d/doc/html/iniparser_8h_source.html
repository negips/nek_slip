171  @param key Key string to look for
172  @param notfound Value to return in case of error

174 
175  This function queries a dictionary for a key. A key as read from an
176  ini file is given as "section:key". If the key cannot be found,
177  the notfound value is returned.
178 
179  Supported values for integers include the usual C notation

186 
187  Warning: the conversion may overflow in various ways. Conversion is

232

235  - A string starting with 'f'
236  - A string starting with 'F'
237  - A string starting with '0'
238 
239  The notfound value returned if no boolean is identified, does not
240  necessarily have to be 0 or 1.
